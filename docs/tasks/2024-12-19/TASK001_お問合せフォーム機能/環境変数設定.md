# 環境変数設定

お問い合わせフォーム機能を使用するには、以下の環境変数を`.env.local`ファイルに設定してください。

## 必要な環境変数

### MongoDB接続設定
```env
MONGODB_URI=mongodb://localhost:27017/ai-code-review
```

または MongoDB Atlas を使用する場合：
```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/database?retryWrites=true&w=majority
```

### SMTP設定（メール送信）
```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
```

**Gmailを使用する場合の注意点：**
- Gmailのアプリパスワードを使用する必要があります
- 2段階認証を有効にしている必要があります
- アプリパスワードの生成方法: https://support.google.com/accounts/answer/185833

### 管理者メールアドレス
```env
ADMIN_EMAIL=admin@example.com
```

お問い合わせを受け取るメールアドレスを設定してください。

## 設定手順

1. プロジェクトルートに`.env.local`ファイルを作成
2. 上記の環境変数をコピーして、実際の値に置き換える
3. `.env.local`ファイルは`.gitignore`に含まれているため、Gitにコミットされません

## 設定例

```env
# MongoDB接続設定
MONGODB_URI=mongodb+srv://user:password@cluster0.xxxxx.mongodb.net/ai-code-review?retryWrites=true&w=majority

# SMTP設定
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=abcdefghijklmnop

# 管理者メールアドレス
ADMIN_EMAIL=admin@example.com
```

## トラブルシューティング

### MongoDB接続エラー
- MongoDBが起動しているか確認
- 接続文字列が正しいか確認
- ファイアウォール設定を確認

### メール送信エラー
- SMTP設定が正しいか確認
- Gmailを使用する場合、アプリパスワードが正しいか確認
- ポート番号が正しいか確認（587または465）
